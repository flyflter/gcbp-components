# 📦 GCBP-Web-Framework - 业务中台前端开发框架
> Web端/移动端开发框架、组件库、类库

## 初始化 所有依赖包 
> 根目录执行 
+ `npm run bootstrap` 安装所有依赖
+ `npm build` 构建所有包 
+ `npm version:auto` 自动构建变更的包，并更新版本号 
+ `npm publish:auto` 自动发布打过新版本号的包
